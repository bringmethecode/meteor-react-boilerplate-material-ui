# Meteor React Boilerplate (MaterialUI styles)

Starter kit for Meteor with React and MaterialUI package.

Unstyled boilerplate here: https://github.com/alonoslav/meteor-react-boilerplate

## Used packages:

#### Npm

- react
- react-dom
- react-mounter
- material-ui
- react-tap-event-plugin

#### Atmosphere

- react-meteor-data
- kadira:flow-router
